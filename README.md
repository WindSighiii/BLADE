## Running Experiments
To run experiments on different datasets, simply execute one of the following scripts:

- `KUA.sh` – for the **KuaiSAR** dataset
- `QKA.sh` – for the **QK-Article** dataset  
- `QKV.sh` – for the **QK-Video** dataset

### Example

```bash
sh KUA.sh   # Run on KuaiSAR
sh QKA.sh   # Run on QK-Article
sh QKV.sh   # Run on QK-Video
